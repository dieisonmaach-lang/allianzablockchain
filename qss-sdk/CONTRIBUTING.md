# Contributing to @allianza/qss-js

Thank you for your interest in contributing to the Allianza Quantum Security Service SDK!

## Development Setup

```bash
# Clone the repository
git clone https://github.com/allianza-blockchain/qss-sdk-js.git
cd qss-sdk-js

# Install dependencies
npm install

# Build
npm run build

# Run tests
npm test

# Watch mode
npm run dev
```

## Code Style

- Use TypeScript
- Follow ESLint rules
- Write tests for new features
- Update documentation

## Pull Request Process

1. Fork the repository
2. Create a feature branch
3. Make your changes
4. Add tests
5. Update documentation
6. Submit a pull request
